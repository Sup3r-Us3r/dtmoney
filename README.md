# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/Sup3r-Us3r/dtmoney.git
```

```bash
# Install Dependencies
$ yarn

# Run Application
$ yarn start
```
Go to http://localhost:3000/ to see the result.
